# Diffusion MRI and Tractography Summer School 
Developed for Centre for Medical Imaging Computing (CMIC)
By Lawrence Binding and Anna Schroder


This website was built to act as a tool for students and clinicians to access lectures and material on diffusion MRI and tractography. This tutorial teaches the basics of diffusion, tractography, how to code your own tractography algorithm, and how to use MRtrix3. 

Access the tools via the following website

[https://diffusion-tractography.github.io/]
